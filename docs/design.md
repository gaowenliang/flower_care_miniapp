# 🪴 养花助手 — UI 设计稿 & 操作流

## 一、设计风格

### 设计关键词
**自然、清新、轻量、治愈**

### 色彩体系
```
主色   #4CAF50  绿色 — 生命力、自然、成长
辅色   #66BB6A  浅绿 — 按钮、高亮
背景   #F7F9F7  米白 — 不刺眼的暖底色
卡片   #FFFFFF  白色 — 纯净
文字   #2D3436  深灰 — 主要文字
次要   #90A4AE  灰色 — 辅助文字
警告   #FF7043  橘红 — 逾期、紧急
成功   #26A69A  青绿 — 已完成
```

### 字体规范
```
页面标题    34rpx  bold   #2D3436
卡片标题    30rpx  600    #2D3436
正文        28rpx  normal #2D3436
辅助文字    24rpx  normal #90A4AE
标签        22rpx  normal
数字强调    40rpx  bold   #4CAF50
```

### 圆角规范
```
卡片圆角    24rpx
按钮圆角    44rpx（全圆角）
```

```
用户点击TabBar"添加" 
  → 进入添加页（显示全部植物列表 + 分类）
  → 两种路径：
      A) 搜索/浏览 → 点击植物 → 弹窗取名+选位置 → 确认 → 自动跳转首页
      B) 点📷拍照 → AI识别（Phase 2）→ 匹配植物 → 弹窗确认
  → 后台自动：创建浇水任务（按植物默认周期）
```

### 4.2 完成任务流程
```
首页点击 ✅ 
  → 【个人模式】storage.completeTask()
  → 【家庭模式】family.completeTask() — 乐观写
     → 立即更新本地缓存（nextDate往后推）
     → 后台串行推云端，失败回滚
  → UI：顶部飘出绿色 toast "完成啦~"
  → 刷新今日任务列表和花园状态
```

### 4.3 逾期提醒机制
```
用户打开小程序 (onShow)
  → subscribe.checkAndNotify()
     → 检查到期任务
     → 有到期任务：wx.showToast 提示
     → 每天首次打开：请求订阅消息授权
  → 云函数定时器：每天9:00 checkReminders
     → 查到期任务 → 推送订阅消息
```

### 4.4 植物详情操作矩阵
```
操作          │  触发位置      │  行为
──────────────┼───────────────┼──────────────────
📷拍照        │  头部按钮      │  打开相机 → 保存为记录
📝备注        │  头部按钮      │  弹出输入框 → 保存为记录
➕任务        │  头部/标题栏   │  弹出类型选择+间隔 → 创建任务
🗑️删除        │  头部按钮      │  二次确认 → 级联删除 → 返回
✓完成         │  任务行右侧    │  完成任务 → 刷新列表
⏸暂停/▶恢复   │  任务行右侧    │  切换enabled状态
−/+间隔       │  任务行中间    │  调整间隔天数(最小1)
Tab切换       │  顶部Tab       │  切换指南/记录/贴士内容
```
