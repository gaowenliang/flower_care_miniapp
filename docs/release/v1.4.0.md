# 🪴 养花助手 v1.4.0 Release Note

**发布日期**：2026-06-14

> 覆盖 v1.3.0（6/1）之后的内容，与 v1.3.0 合计 128 commits / 20 天

---

## 🆕 新功能

### 🌧️ 补浇水 / 逾期提醒
- 植物详情支持补浇水（补卡记录），本月限 3 次
- 逾期任务持续显示逾期天数，让用户知道拖了多久

### 📸 截图导入养护记录
- 日历页支持从其他 APP 截图导入养护记录
- OCR + 正则解析，兼容日历网格等复杂格式
- 支持相对时间（X 天前 / X 个月前 / X 年前）推断年份
- 可选择植物和操作成员

### 🪦 植物嘎了功能
- 详情页 💀 标记嘎了 → 头像变灰 → 自动移到天堂房间
- 可复活，恢复原房间
- 天堂房间的植物不显示养护提醒

### 🌡️ 天气功能
- 首页显示天气 + 下雨天提醒不浇水
- 云函数 + 前端直连双通道兜底
- 天气获取失败显示错误原因 + 点击重试

### 🏠 房间管理
- 房间支持修改名称和删除（含预设房间）
- 房间一键浇水、全房间浇水
- 房间环境参数改为 5 档点选按钮

### 📅 日历增强
- 日历页显示家庭所有成员的养护记录（带操作人昵称）
- 养护记录改为热力图视图 → 胶囊时间线
- 胶囊按养护类型显示不同颜色

### 🧪 施肥类型记录
- 完成施肥时可选择肥料类型（快捷 + 自定义输入）
- 肥料类型写入养护记录

### 📝 植物日记增强
- 支持多张照片同时上传
- EXIF 日期比 addedAt 更早时自动更新
- 日记照片和备注支持点击删除

### 🔍 植物分类识别
- 详情页学名处放大镜 → 支持分类搜索
- 种类 + 科名双匹配，种类优先显示

### 💰 花费统计
- 增加设备购买和废料购买记录按钮
- 按科分布 bar 最多的顶满，其他按比例缩放

### 👋 新用户引导
- 家庭页步骤指示器
- 首页引导添加植物
- 创建/加入家庭后自动跳转首页

### 📱 UI 优化
- 植物详情页头部信息层级重排 + 头像放大 + 操作按钮优化
- 弹窗改为居中卡片样式（所有页面统一）
- 详情页养护记录从日历视图 → 胶囊时间线
- 首页头像放大 + 圆角矩形 + 3/4 列布局
- 输入框高度统一 88rpx
- 到家日期可编辑 + 显示养了多少天

## 🛠 重构 / 优化

### plant-detail.js 拆分
- 1170 行 → 317 行（-73%）
- 提取 3 个 behavior：
  - `task-manager-behavior.js` — 完成任务、添加任务、调整周期
  - `plant-editor-behavior.js` — 编辑昵称/价格/位置/头像、嘎了/复活

- 55 个重复样式块提取到 app.wxss
- 7 个页面瘦身

- 所有页面 add setTimeout → `_timer` + `onUnload` 统一清理（7 个页面）

- profile.js 竞态防护（_loadId 版本号）
- storage.js 新增数据迁移机制（v0→1→2）
- 天气缓存全局 + 按天去重 + 持久化到 Storage

### 代码清理
- 删除 3 个死代码文件（theme.js / network.js / placement.js）
- 删除 app.js 旧接口残留
- console.log → console.debug
