# 🪴 养花助手 v1.3.0 Release Note

**发布日期**：2026-06-01 · **Commit 数**：50

---

## 🎉 新功能

### 养护记录大改版
- 植物详情养护记录从日历视图改为**胶囊时间线**视图
- 胶囊按养护类型显示不同颜色：浇水蓝 / 施肥橙 / 修剪粉 / 换盆棕
- 胶囊始终显示养护人昵称，支持长按删除
- 新增**日历热力图**视图，按周展示，图标+成员颜色

### 房间管理
- 房间支持修改名称和删除（含预设房间）
- 预设房间改名后不再显示原名
- 待办推迟按钮

### OCR 批量导入增强
- 支持相对时间解析（X天前/X个月前/X年前）
- 三重年份保障 + 预扫描年份
- 支持选择养护人 + 重复记录覆盖更新

### 其他
- 日记照片和备注支持点击删除
- 植物详情拍照支持多张上传
- 分类搜索支持种类+科名双匹配
- 植物详情页可编辑到家日期

---

## 🐛 修复

- 养护记录查询改用 fetchAll 分页，解决 2025 年前数据截断
- 删除记录真正持久化（修复 id 字段不匹配）
- input 输入框高度统一 88rpx，文字不再被截断
- 首页头像提示数字被裁切
- getWeather 改用 https
- importScreenshot 年份解析多处修复（19个日期被吞等）
- 日历导入失败改 showModal 显示完整错误
- 到家日期改用 addedAt 字段
